中高層集合住宅の木造・混構造化について、

構造・遮音・財務の3軸で要点を整理する。

---

## 1. 混構造（ハイブリッド）の設計ロジック

下層部（1〜3F）：RC造  →  重量・耐火・防火地域対応

上層部（4〜7F）：木造  →  軽量化・意匠・工期短縮

**軽量化のカスケード効果：**

- 上層を木造（RC比1/5の重量）にすると地震力が低減
- 下層RC部材の断面縮小が可能
- 基礎（杭）の本数・深さを削減 → 土工事コスト圧縮

**適用ルート：**

- 2025〜2026年の構造計算プログラム更新で混構造ルートが整備済み
- 日本建築学会「混構造設計指針」を参照

## 2. 遮音設計：CLTオーバーレイ工法の基本構成


| 階層 | 主な材料 | 役割 |
| ------ | ---------------------- | -------------------- |
| 構造層 | CLTパネル（90〜150mm） | 床全体の剛性を確保し、水平力を伝達する |
| 制振・遮音層 | 高減衰ゴム、ALC、アスファルト系遮音シート | 振動を減衰させ、重量床衝撃音を抑制する |
| 吸音・断熱層 | グラスウール、ロックウール（天井懐充填） | 空気伝播音を吸収し、断熱性能を向上させる |


この構成は、中高層木造マンションで採用される「高遮音ハイブリッド床システム」の典型例です。CLTで構造性能を確保し、その上に重量層と制振層を設け、さらに天井内の吸音材で空気音を処理することで、RC造に近い居住性能を実現します。

BIMによる積層シミュレーションで事前検証可能。

設計初期段階での床構成確定が重要。

## 3. 財務比較：RC造 vs 木造ハイブリッド（4F・延床1,000㎡・30戸 試算）

|


| 比較項目 | RC造 | 木造ハイブリッド |
| ---------- | -------- | --------------- |
| 建築コスト（坪単価） | ～200万円 | ～130万円 |
| 工期 | 18～24ヶ月 | 9～12ヶ月 |
| 法定耐用年数 | 47年 | 22年（木造部） |
| ESG・環境価値 | 低い | 高い（炭素固定量を可視化可能） |
| 主な特徴 | 長寿命・実績豊富 | 低コスト・短工期・脱炭素対応 |


※坪単価や工期は建物規模・地域・仕様・RCコア比率によって大きく変動するため、あくまで一般的な目安です。

※ AQ Group社発表実績・業界標準値をもとにした参考試算。

  地域・地盤・設計条件によって変動する。

**木造ハイブリッドの財務上の優位点：**

1. **早期稼働** ── 工期短縮でIRR改善
2. **減価償却加速** ── 木造22年で節税効果（投資家・法人向け）
3. **材料価格の安定性** ── 国産材＋地域アライアンスで調達リスク低減
4. **ESG融資** ── BIM×LCAで炭素固定量を数値化し低利融資へ
